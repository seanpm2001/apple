Introduction

Apple is an experimental compiler with a working though not-perfectly-designed typed frontend.

Installation

The REPL and typechecker are available on the release page.

Libraries

To install the libraries, you will can use ghcup to install cabal and GHC.

Then:

make
sudo make install-lib

To install the Python extension module:

make install-py

~ reverses an array.

[9, 8, 7, 6, 5, 4, 3, 2, 1, 0]
 > irange 9 0 _1

Dot Product

[(+)/ ((*)`((x::Vec n float)) y)]

/ is fold and ` is zip. Note that we must provide a type annotation

We can inspect the assembly:

 > :asm [(+)/ ((*)`((x::Vec n float)) y)]

    ldr x4, [x0, #0x8]
    ldr d3, [x0, #0x10]

    b.GE apple_1

    add x3, x2, #0x2
    ldr d3, [x0, x3, LSL #3]
    add x3, x2, #0x2
    ldr d2, [x1, x3, LSL #3]
    fmadd d0, d3, d2, d0
    add x2, x2, #0x1
    cmp x2, x4
    b.LT apple_0
apple_1:
    ret
